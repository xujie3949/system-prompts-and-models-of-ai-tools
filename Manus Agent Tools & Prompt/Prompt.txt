# 高效提示词指南

## 提示词编写导论

本文档提供与AI助手协作时的提示词编写指南。精心设计的提示词能显著提升响应质量和相关性。

## 高效提示词关键要素

### 明确具体
- 直述请求内容
- 包含相关背景信息
- 指定响应格式要求
- 说明限制条件

### 提供上下文
- 解释需求原因
- 分享相关知识背景
- 如适用说明先前尝试
- 描述自身对该主题的了解程度

### 结构化请求
- 将复杂请求拆分为小模块
- 多部分问题使用编号列表
- 多项请求时标明优先级
- 考虑使用标题分段组织

### 指定输出格式
- 说明期望响应长度（简略/详细）
- 要求特定格式（要点/段落/表格）
- 如需代码示例/引用等特殊元素需声明
- 如相关需指定语气风格（正式/口语/技术）

## 提示词示例

### 欠佳提示：
"告诉我机器学习相关知识"

### 优化提示：
"我是计算机科学专业学生，正在开展首个机器学习项目。能否用2-3段文字解释监督学习算法，重点说明图像识别中的实际应用？请包含2-3个具体算法案例及其优缺点。"

### 欠佳提示：
"写个网站代码"

### 优化提示：
"需要为个人作品集网站创建简易联系表单。能否编写响应式的HTML/CSS/JavaScript代码，收集姓名/邮箱/留言字段？表单提交前需验证输入，设计需符合蓝白配色的极简美学。"

## 迭代式提示

与AI助手协作通常是迭代过程：

1. 初始提示
2. 评估响应
3. 根据有效/缺失内容优化提示
4. 持续对话深入探讨

## 代码相关提示

请求代码示例时建议包含：

- 编程语言及版本
- 使用的库/框架
- 如调试需提供错误信息
- 输入输出示例
- 性能考量
- 兼容性要求

## 结语

高效提示是项需练习的技能。通过清晰、具体且提供背景的提示，您能从AI助手获得更具价值的响应。若初始回复未完全满足需求，可随时优化提示词。
